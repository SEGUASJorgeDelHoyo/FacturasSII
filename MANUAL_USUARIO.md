# Manual de Usuario - Facturas SII

Este manual explica con pasos sencillos cómo usar la aplicación sin necesidad de conocimientos técnicos.

## 1. Abrir la aplicación

1. Busca el archivo `start_app.bat` dentro de la carpeta de la aplicación.
2. Haz doble clic en `start_app.bat`.
3. Se abrirá una ventana del sistema y, en unos segundos, se mostrará la aplicación en tu navegador.
4. Si el navegador no se abre solo, abre el navegador y ve a `http://localhost:3000`.

> Importante: deja abierta la ventana que aparece al iniciar. No la cierres hasta terminar de trabajar.

## 2. Escribir la factura

1. En la aplicación, localiza el campo de texto que dice "Referencia de Factura (Odoo)".
2. Escribe el nombre exacto de la factura proveedor tal como aparece en Odoo.
3. Ejemplo de nombre: `BILL/2026/03/0001`.
4. No uses palabras sueltas ni datos incompletos: debe ser la referencia exacta.

## 3. Buscar la factura

1. Haz clic en el botón `BUSCAR EN ODOO`.
2. Espera unos segundos.
3. Si la factura existe, verás sus datos en pantalla:
   - Referencia
   - Estado SII
   - Proveedor
   - Fecha
   - Importe total
4. Si no encuentra una coincidencia exacta, te mostrará facturas similares para ayudarte.

## 4. Comprobar y actualizar la factura

1. Cuando la factura aparezca en pantalla, revisa que los datos sean correctos.
2. Haz clic en el botón `ACTUALIZAR ESTADO SII`.
3. Espera el mensaje que confirma que la factura se actualizó correctamente.
4. Si ves un error, revisa la referencia y vuelve a intentarlo.

## 5. Cerrar la aplicación

1. Puedes cerrar la aplicación usando el botón `Cerrar Aplicación` en la pantalla.
2. Si no puedes cerrar desde la aplicación, cierra la ventana del navegador y luego la ventana del sistema.

## Consejos rápidos

- Usa siempre el nombre de factura completo y exacto.
- Si no estás seguro, copia el nombre desde Odoo y pégalo en la aplicación.
- No necesitas cambiar ninguna configuración ni tocar otros archivos.

## ¿Y si no encuentra la factura?

- Prueba escribir la factura completa otra vez.
- Revisa si hay espacios extras antes o después del texto.
- Si aún no aparece, pregúntale a quien gestiona las facturas en Odoo para confirmar la referencia exacta.
